# 多仓库版本控制
> 从官方仓库fork了一个仓库，但是想要做自己的修改

1. 获取项目代码: `git clone <official_repo_url>`
2. 增加个人上游仓库: `git remote add <remote_name> <git_url>`
3. 重命名: `git remote rename <old_name> <new_name>`
